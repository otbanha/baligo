Hari ini, 17/8 siang, terjadi kebakaran di basement Pasar Ubud. Api berkobar dari siang hingga sore belum juga padam. Kebakaran ini menyebabkan 1.033 pedagang di pasar tidak bisa berjualan.

Api mulai menyebar ke basement pasar sekitar pukul 13.00 waktu setempat. Sampai sore, lebih dari sepuluh mobil pemadam kebakaran sudah dikerahkan ke lokasi, tapi api masih belum bisa dikendalikan.

Sekretaris Daerah Gianyar, I Dewa Gede Alit Mudiarta, langsung turun ke lokasi untuk memantau proses pemadaman. Dia bilang, 'Kami sedang berusaha keras memadamkan api. Prioritas utama adalah memastikan tidak ada korban jiwa, baru kemudian menangani kerugian barang dagangan para pedagang. Karena proses pemadaman masih berlangsung, kami belum bisa memastikan kerugian secara detail.'

Berdasarkan data yang ada, total ada 1.033 pedagang di pasar ini. Sekitar 400 orang di antaranya berjualan di basement (tempat asal api), dan mereka adalah yang paling terdampak langsung.

Dewa Alit menambahkan, 'Soal penempatan pedagang dan masalah lainnya, akan kita bahas dalam rapat. Untuk sekarang, fokus kami adalah memadamkan api secepat mungkin dan mencegahnya menjalar ke area lain.'

Sampai sekarang, penyebab kebakaran masih dalam penyelidikan. Kabarnya, api diduga berasal dari area parkir motor di basement.
