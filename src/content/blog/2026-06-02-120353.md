### 🗞️ 今日要聞：觀光投資去中心化與數位執照驗證系統

- **印尼觀光部長發出呼籲：投資應超越峇里島四大熱區**：
印尼觀光與創意經濟部長維迪揚蒂（Widiyanti Putri Wardhana）今日在島內舉行的「2026 投資者圓桌會議」上發表核心演說。她指出，目前的觀光投資過度集中於南部與熱門區域，包括**倉谷（Canggu）、塞米亞克（Seminyak）、烏魯瓦圖（Uluwatu）以及烏布（Ubud）**。為了推動全印尼更均衡且具再生性的經濟增長，政府正優先發展 10 個國家觀光優先目的地，極力邀請國際與在地資本將眼光投向這些尚未被過度開發的潛力區域。
- **官方整頓線上非法旅宿：將推出 API 數位執照驗證系統**：
為了應對大量未經合法的民宿與別墅直接上架國際訂房平台所引發的逃漏稅及消費糾紛，觀光部今日同時宣布，**目前正在加緊籌備一套「基於 API 的數位執照驗證系統（API-based verification system）」**。未來這套系統將強制與各大跨國數位訂房平台對接，確保所有上架營運的峇里島住宿皆百分之百符合印尼官方的商業執照與安全規範。

### 📄 遊客關心話題：網路執法破獲走私與六月文創盛會

- **Facebook 露馬腳！網警巡邏偵破吉安雅（Gianyar）傳統藝品店象牙走私案**：
印尼林務部執法總局（Gakkumhut）今日在登巴薩召開聯合通報，正式宣告破獲一起利用社群媒體販售受保護野生動物製品的跨境大案。執法小組在進行線上網路巡邏（Cyber Patrol）時，發現 Facebook 上有人公開兜售疑似保育類動物製品。警政人員隨即順藤摸瓜直搗**吉安雅縣聖泉寺（Tampaksiring）周邊的兩家傳統藝品店（Art Shop）**，當場查獲大批珍貴的**象牙手工雕刻品與蒐藏工藝品**。涉案嫌犯 IKS 已正式遭到逮捕並移送法辦，官方藉此重申對任何破壞生態保育的觀光商業犯罪零容忍。
- **島內六月最大本土盛會：第八屆「**[**蘇卡諾月（Bulan Bung Karno）**](https://blitarkota.go.id/infografis/bulan-bung-karno)**」於藝術中心揭幕**：
由峇里島省長與副省長主持，年度規模最盛大的本土文化與歷史慶典——**第八屆「蘇卡諾月」於今日在登巴薩 Ardha Candra 露天舞台（Art Center）正式宣告拉開序幕**。在整個六月份，省政府將在島內各社區與文藝中心，舉辦一系列涵蓋創意設計、歷史教育、傳統工藝與在地美學的文創競賽與展演。這項活動旨在傳承建國先驅的文化精神，也為六月抵達島內、喜愛人文探索的自由行旅客提供了極佳的沉浸式文化體驗機會。（蘇卡諾是印尼國父）
